# Abstract Games Website

Абстрактный игровой сайт с счётчиком посетителей в реальном времени.

## Деплой на Vercel

### Способ 1: Через Vercel CLI

1. Установите Vercel CLI:
```bash
npm i -g vercel
```

2. Войдите в аккаунт:
```bash
vercel login
```

3. Задеплойте проект:
```bash
vercel
```

4. Для продакшена:
```bash
vercel --prod
```

### Способ 2: Через GitHub

1. Создайте репозиторий на GitHub
2. Загрузите код в репозиторий
3. Перейдите на [vercel.com](https://vercel.com)
4. Нажмите "New Project"
5. Импортируйте ваш GitHub репозиторий
6. Vercel автоматически определит настройки и задеплоит проект

### Способ 3: Через веб-интерфейс Vercel

1. Перейдите на [vercel.com](https://vercel.com)
2. Нажмите "Add New Project"
3. Загрузите папку проекта через drag & drop
4. Vercel автоматически задеплоит проект

## Структура проекта

- `index.html` - главная страница
- `style.css` - стили в серых оттенках
- `counter.js` - клиентский скрипт для счётчика
- `api/counter.js` - API endpoint для счётчика посетителей
- `vercel.json` - конфигурация Vercel

## Особенности

- Абстрактный дизайн в серых оттенках
- Счётчик посетителей в реальном времени (обновляется каждые 3 секунды)
- Анимированные элементы и частицы
- Адаптивный дизайн

## Примечание

Счётчик использует in-memory хранилище, которое работает в рамках одного инстанса Vercel. Для продакшена рекомендуется использовать Vercel KV или внешнюю базу данных для более точного подсчёта между несколькими инстансами.
