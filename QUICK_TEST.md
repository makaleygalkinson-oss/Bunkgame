# Быстрый тест - Почему не работает кнопка "Создать"

## Шаг 1: Проверка консоли

1. Откройте сайт
2. Нажмите **F12**
3. Перейдите на вкладку **Console**
4. Нажмите кнопку "Создать Лобби"
5. **Что выводится в консоли?**

Ожидаемые сообщения:
```
=== CREATE LOBBY BUTTON CLICKED ===
Current createModalOpen state: false
setCreateModalOpen(true) called
CreateLobbyModal isOpen: true
CreateLobbyModal: rendering
```

## Шаг 2: Проверка модального окна

После нажатия кнопки "Создать Лобби":
- **Появляется ли тёмное окно с формой?**
- Если ДА → переходите к шагу 3
- Если НЕТ → проблема в открытии модального окна

## Шаг 3: Заполнение формы

1. Введите имя игрока
2. Введите название комнаты
3. Выберите роль
4. Нажмите "Создать"

**Что происходит?**
- Появляется ли сообщение "Создание..."?
- Есть ли ошибки в консоли?
- Что выводится в консоли при нажатии?

## Шаг 4: Проверка Network

1. В DevTools перейдите на вкладку **Network**
2. Попробуйте создать лобби
3. Найдите запрос к `/api/game/create`
4. Посмотрите:
   - **Status** (200 = успех, 400/500 = ошибка)
   - **Response** (что вернул сервер)
   - **Request Payload** (что отправили)

## Возможные проблемы

### Проблема 1: Кнопка не реагирует
**Решение:** Проверьте консоль на ошибки JavaScript

### Проблема 2: Модальное окно не открывается
**Решение:** 
- Проверьте, что нет ошибок в консоли
- Проверьте стили (возможно z-index конфликт)
- Попробуйте в другом браузере

### Проблема 3: Форма не отправляется
**Решение:**
- Проверьте, что все поля заполнены
- Проверьте консоль на ошибки валидации
- Проверьте Network tab

### Проблема 4: Ошибка 500/400
**Решение:**
- Проверьте переменные окружения
- Проверьте подключение к Supabase
- Проверьте логи сервера
